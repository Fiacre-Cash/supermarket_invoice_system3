# Supermarket Invoice System

A lightweight web-based supermarket invoice and billing system.

## Features

- Static HTML/CSS/JS frontend
- Simple server entry point `server.js`
- Invoice, billing, products, sales and team pages

## Quick start

1. Install Node.js (v14+ recommended)
2. From the project root run:

```bash
# (optional) install dependencies if you use any
# npm install

# Start the server
node server.js
```

3. Open your browser at http://localhost:3000 (or the port configured in `server.js`)

## Development

- Frontend files live in the project root and `assets/`, `css/`, `js/` folders
- Edit HTML files and reload the browser to see changes
