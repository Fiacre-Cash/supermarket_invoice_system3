2. LOGIN CREDENTIALS:
   - Admin Account:
     * Username: admin

   - Cashier Accounts:

       Password: cashier123

3. FEATURES:
   - Admin Panel:
     * Manage products (add, edit, delete)
     * View sales history
     * Generate reports
     * Manage user accounts
     
   - Cashier Panel:
     * Create new invoices
     * Process customer purchases
     * View recent transactions

4. DATA INITIALIZATION:
   - The application automatically initializes with sample data when loaded
   - Sample products, users, and invoices are created
   - All data is stored in your browser's localStorage

5. NAVIGATION:
   - After login, you'll be redirected to the appropriate dashboard
   - Use the navigation menu to access different sections
   - Click 'Logout' to exit the system

For any issues, check the browser console for error messages.
